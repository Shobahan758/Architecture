# Nexora - Architecture & Interior Design HTML5 Template

Thank you for purchasing Nexora. Nexora is a premium, responsive Bootstrap 5 HTML template for architecture studios, interior designers, construction companies, furniture studios, renovation businesses, real estate interiors, and creative agencies.

## Important Note

Nexora is a static HTML template. It is not a WordPress theme and cannot be installed through the WordPress dashboard.

## Included Files

- 7 homepage demos: `index.html`, `home-02.html`, `home-03.html`, `home-04.html`, `home-05.html`, `home-06.html`, and `home-07.html`
- Inner pages: About, Services, Service Details, Projects, Project Details, Team, Team Details, Pricing, FAQ, Blog, Blog Details, Contact, Shop, Product Details, Cart, Checkout, and 404
- Assets: `assets/css/main.css`, `assets/css/responsive.css`, `assets/js/main.js`, organized images, local fonts, Bootstrap, and Bootstrap Icons
- Documentation: `documentation/index.html`
- Extra notes: `credits.md` and `changelog.md`

## Installation Guide

1. Unzip the package.
2. Open `index.html` in your browser for a quick preview.
3. Choose the homepage and inner pages you want to use.
4. Upload the selected HTML files and the complete `assets` folder to your hosting server.
5. Keep the folder structure unchanged unless you also update all related asset paths.

## Used Fonts

- Inter: body text, navigation, buttons, forms, and UI labels
- Playfair Display: hero headings, section headings, editorial titles, and brand typography

Fonts are loaded locally from `assets/vendor/fonts/google-fonts.css`.

## Used Plugins

- Bootstrap 5: responsive grid, navbar collapse, dropdowns, accordions, and form styling
- Bootstrap Icons: icon set used across buttons, cards, social links, and UI controls
- Custom JavaScript: theme toggle, sliders, filters, counters, reveal animation, and form demo behavior

No premium plugin is required.

## Customization Guide

Edit content directly inside the HTML files. Custom styles are located in:

- `assets/css/main.css`
- `assets/css/responsive.css`

Main colors can be changed from CSS variables in `assets/css/main.css`.

```css
:root {
  --accent: #b78a54;
  --dark: #171512;
  --soft: #f4efe8;
}
```

## Forms
